How to establish connection:

1. Copy and paste colab_setup.py into a new cell block in Google Colab.
2. Make sure you select a T4 GPU Runtime inside Colab settings.
3. Execute the block. It downloads your custom GGUF file and exports your tunnel URL.
4. Copy the secure .trycloudflare.com URL and paste it in the Settings panel of this site.
