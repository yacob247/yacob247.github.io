Privacy and Cookies

Envizion Tools is a collection of browser utilities for local file preparation, conversion, previewing, and editing. This page explains the privacy model in plain language so visitors can decide whether a tool is appropriate for their files.

Local file processing

Most tools are designed to process selected files inside the browser using local APIs such as Canvas, WebGL, Web Audio, media codecs, PDF libraries, and JavaScript. The intended workflow is that files stay on your device while the page creates previews or downloads.

Some pages load third-party open-source libraries from content delivery networks to make conversion, rendering, or preview features work. Browser support, library behavior, and device memory can affect the result, so sensitive files should be tested carefully before use.

Advertising and cookies

Pages may display Google AdSense advertising. Google and its partners may use cookies, identifiers, web beacons, IP addresses, and similar technologies for ad serving, measurement, fraud prevention, personalization where allowed, and frequency capping.

Visitors can learn how Google uses information from sites and apps that use its services at Google's partner sites policy page.

What you should not upload or process

Do not use these tools for files you are not allowed to process, files that violate someone else's rights, or material that would break publisher, platform, or local legal requirements. Keep originals until you verify exported files.

Contact and corrections

If a page description is inaccurate, a tool behaves differently than described, or an advertising/privacy disclosure needs correction, use the contact page and include the page name, browser, and a concise description of the issue.
