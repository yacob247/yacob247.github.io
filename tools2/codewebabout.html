Mission
Why envizionwork exists

Most developer tooling is either locked behind a paywall, requires a cloud account, or needs a 200MB install to do something simple. envizionwork is the opposite.

Every tool runs entirely in your browser. Files never leave your machine. There's no sign-up, no subscription, and no tracking. Open the page, use the tool, close the tab — that's it.

The suite covers the full breadth of everyday dev work: editing and converting code, compositing video and images, previewing 3D assets, running local servers, and working through terminal guides for Docker, SSH, AI models, databases, and more.

The goal is simple: give developers fast, private, capable tools that work the moment you need them.

Tech stack
Built with
📐

Vanilla HTML / CSS / JS

No frameworks, no bundlers. Each tool is a single self-contained file that loads instantly on any device.

🧊

Three.js

WebGL-powered 3D rendering for the OmniViewer, Animator Studio, and browser-based game experiences.

🗄️

IndexedDB & File System API

Local-first storage for the Studio editor. Assets stay on your machine — no cloud dependency.

🎬

MediaRecorder API

In-browser video export from the compositor editor — no FFmpeg install, no server round-trip.

🔤

Monaco Editor

VS Code's editor engine embedded in the OmniViewer for full syntax highlighting and IntelliSense.

🚀

GitHub Pages

Deployed on GitHub Pages for zero-latency global delivery with no server infrastructure to maintain.
