Image optimizer guide

Batch upscale, compress, and export images.

Image Optimizer Pro is built for people who need smaller images without sending files to a server. Add images, choose Auto-format to keep each image as PNG, JPG, or WebP, pick an upscale pass, set a final width, and process the batch locally.

  • Use Auto-format when you want a mixed batch to stay mixed: PNG files stay PNG, JPG files stay JPG, and WebP files stay WebP.
  • Use WebP for small modern web images, JPEG for photos, and PNG when you need transparency or lossless output.
  • The folder button saves unzipped files directly in supported Chromium browsers. Other browsers will fall back to individual downloads.
  • The ZIP export uses a browser ZIP builder so image batches can be uploaded, archived, or shared as one file.
