How Local Conversion Works

  1. Your computer loads the MP4 container locally via File API.
  2. The Web Audio API decodes the audio streams into high-fidelity float arrays.
  3. The background Web Worker Thread converts raw float arrays into standardized 16-bit PCM buffer blocks.
  4. An optimized lame.js instance bundles MP3 frames in parallel.
  5. Your browser packages everything into an offline MP3 Blob download.
