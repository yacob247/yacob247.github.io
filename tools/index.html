How the Tool Flow Works

The directory is intentionally split into explanation pages and working pages so users know what will happen before opening a heavier tool.

1

Read the tool home

Each home page explains purpose, best use cases, and the basic steps in short cards.

2

Launch the tool

The working tool opens in the same flow, with navigation back to the directory and related tools.

3

Keep files local

Most tools process files in the browser session. Avoid clearing site data if you need saved local workbench files.
