Video watermark guide

Add animated protection before publishing videos.

This Envizion video watermark studio renders editable watermark layers onto a canvas and records a new video. Start with one watermark, drag it into place, change its text, colors, font, size, rotation, opacity, animation, and timing, then click Add Watermark if you want another independently controlled layer.

  • Click a watermark in the preview or select it from the layer list to edit that layer only.
  • Use start and end times for intros, lower thirds, creator tags, paid asset previews, and brand marks.
  • Rendering happens in real time, so a one-minute video usually takes about one minute to export.
  • Exports use WebM where supported by the browser. Audio track support depends on browser media capture features.
